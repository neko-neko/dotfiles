# My dotfiles
[prezto](https://github.com/sorin-ionescu/prezto) base my dotfiles

# Installation
1. Install my dot files:  
    ```
    zsh -c "$(curl -s https://raw.githubusercontent.com/neko-neko/dotfiles/master/install.zsh)"
    ```

# Uninstallation
1. run this:  
    ```
    cd ~/.dotfiles && zsh uninstall.zsh
    ```
