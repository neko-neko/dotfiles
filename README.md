my dotfiles

# Installation
1. Install XCode
2. Install my dot files:  
    ```sh
    zsh -c "$(curl -s https://raw.githubusercontent.com/neko-neko/dotfiles/master/setup/setup.zsh)"
    ```

# Uninstallation
1. run this:  
    ```
    cd ~/.dotfiles && ./setup/uninstall.zsh
    ```
