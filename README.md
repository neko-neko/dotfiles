# My dotfiles
[prezto](https://github.com/sorin-ionescu/prezto) base my dotfiles

# Installation
1. Install my dot files:  
    ```
    curl https://raw.githubusercontent.com/neko-neko/dotfiles/master/install.zsh | zsh
    ```

# Uninstallation
1. run this:  
    ```
    cd ~/.dotfiles && zsh uninstall.zsh
    ```
