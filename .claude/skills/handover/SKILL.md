このセッションで行った作業を振り返り、現在のプロジェクトの `.claude/handover.md` に以下のフォーマットで書き出すこと。
`.claude/` ディレクトリが存在しない場合は作成すること。

```
# Session Handover
> Generated: {現在時刻を ISO8601 形式で出力}

## 作業内容
{完了したタスクと未完了のタスクを箇条書き}

## 未完了タスク
{次にやるべきことを優先順位順に箇条書き。なければ「なし」}

## 決定事項
{採用した方針とその理由を箇条書き}

## 問題と解決策
{遭遇した問題とその解決方法を箇条書き。なければ「なし」}

## 教訓・注意点
{今後注意すべきポイントを箇条書き。なければ「なし」}

## 重要ファイル
{変更・参照した主要ファイルのパスと役割を箇条書き}
```

制約:
- 200行以内
- コードブロックは含めず、ファイルパス:行番号で参照
- このセッションで実際に行った事実のみ記述し、推測や補足は含めない
- 日本語で出力
